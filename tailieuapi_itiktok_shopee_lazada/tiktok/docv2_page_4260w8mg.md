# What is changing?

From June 17, 2024, Platform updated the collection method to support sellers to choose different collection methods for different warehouse content. The parameter "handover\_method" in API [Ship Package](<https://partner.tiktokshop.com/docv2/page/650aa4f1defece02be6e7cb1?external_id=650aa4f1defece02be6e7cb1#Back To Top>) and [Batch Ship Package](<https://partner.tiktokshop.com/docv2/page/650aa3ebbace3e02b75d8bd4?external_id=650aa3ebbace3e02b75d8bd4#Back To Top>) will be useless from March 31, 2026.

# Which markets are affected?

The updates of the requirements apply to cross-border overseas sellers in Singapore, Philippines, Vietnam, Thailand and Malaysia markets.

# Who is affected?

Developers with applications that use [Ship Package](<https://partner.tiktokshop.com/docv2/page/650aa4f1defece02be6e7cb1?external_id=650aa4f1defece02be6e7cb1#Back To Top>) and [Batch Ship Package](<https://partner.tiktokshop.com/docv2/page/650aa3ebbace3e02b75d8bd4?external_id=650aa3ebbace3e02b75d8bd4#Back To Top>).

# Which version is applicable?

The updates of the requirements apply to all versions of the related APIs

# What action is required?

Developers need to check whether to use handover\_method to update the collection method. It will be useless.  
If sellers want to know the order's collection method, they need to go to Seller Center to see the setting of related warehouses. The path is "Orders"->"Manage Orders"->"Fulfillment settings"->"Collection method settings".
