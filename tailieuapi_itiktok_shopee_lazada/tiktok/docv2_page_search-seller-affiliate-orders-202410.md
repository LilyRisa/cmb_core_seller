Search Seller Affiliate Orders

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.affiliate\_collaboration.read

вЂ‹

This API allows the partner to retrieve a list of affiliate orders (orders which are affiliate-commission eligible) generated by a seller, returning the order ID and the product ID. Using this, the partner can track their affiliate-conversions on behalf of a seller, using the order ID.вЂ‹

Filter orders to show only those that are created on or after the specified date and time. Unix timestamp.вЂ‹

Note:вЂ‹

create\_time\_ge and create\_time\_lt together constitute the creation time filter condition.вЂ‹

\- If create\_time\_ge is filled but create\_time\_lt is empty, create\_time\_lt will default to the current time.вЂ‹

\- If create\_time\_lt is filled but create\_time\_ge is empty, create\_time\_ge will default to the earliest shop time.вЂ‹

If both are not specified, data from the last three months will be queried by default.вЂ‹
