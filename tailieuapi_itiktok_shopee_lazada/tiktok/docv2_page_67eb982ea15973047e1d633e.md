# Introduction: What is Creator Leads Management？

Creator Leads is a feature designed for CAP to manage and follow up with all creator leads before establishing an official linking relationship with the creator.  
++Please note: This feature is still under testing and is only available to CAPs on the allowlist. Please check with your CAP manager if you have any questions.++

# What are the benefits？

✅ **More Creator Leads:** you can spread the promotion code offline to connect with more creators.  
✅ **Centralized Lead Management:** Provide a consolidated page to handle and track all livestream creator leads

### Check creator performance to consider further collaboration.

You can search specific creators by **username,** and **view** the creator profile by clicking the creator's name.

-   **Creator level:** Multi-select tiers (Bronze, Silver, Gold, Platinum, Ruby, Emerald, Sapphire, Diamond) to focus on specific creator levels. Tier data refreshes on the 2nd day of each month.
-   **Follower count:** Select predefined ranges (0–10K, 10K–100K, 100K–1M, 1M–10M, 10M+) or manually enter values. Data refreshes daily.
-   **Lead Source:** See if a lead comes from "Collaboration request" (formal submission) or "Instant message" (marked by you).

![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/133b3574f66741c49b94e5b94be24dc0~tplv-k9wyc2ijk0-image.image)  
***Note*:** Creator data authorization expires after 7 days for formal collaboration requests, so you should act promptly to follow up.  
You can check the highest GMV LIVE dashboard in the last 30 days by clicking View Live Dashboard.  
Note:

1.  There will be no data if this creator has no live stream data in the past 30 days  
    2)The visible data is valid for only 7 days. If it has been 7 days since the creator applied, the information will no longer be accessible.

### Message Creator and invite to link

You can send a message to the creator to get more information, and share invitations to link with this creator if you would like to build an official linking relationship with this creator.  
Note:  
The "invite to link" button may not appear if this creator is already linked with you.

![](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/518c12ef1e294d348e844985071f78a6~tplv-k9wyc2ijk0-image.image)

### Manage your creator leads with status tags and notes.

Status tag: The lead status will be set as a new opportunity as default. You can update the lead's status manually according to your needs.

-   **Follow up:** All newly created leads.
-   **In progress:** Leads currently in contact progress.
-   **Contacted:** Leads that have been successfully contacted.
-   **Invalid**: Leads that are no longer valid or actionable.

![](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/d24ff98fc76f45869d3f6ba3a0221aa2~tplv-k9wyc2ijk0-image.image)

You can also add notes (up to 200 words) for leads to help you manage them more effectively.

### FAQ

### Why can't I access leads management?

Lead management is now only available for the cap on the allowlist. Please contact your account manager for more information.

### How can I be displayed on this Top LIVE agencies list？

Please get in touch with your account manager and check if you have met the criteria for Top LIVE agencies CAP

### Why can't I see part of the creator's data or contact info？

There are two possible reasons:  
1)The data might not be available for display, or the creator may not have authorized data sharing.  
2)The visible data is valid for only 7 days. If it has been 7 days since the creator applied, the information will no longer be accessible.
