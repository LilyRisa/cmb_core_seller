Recommend Category

‚ÄčShop

‚Äč

Required scope:

‚Äčseller.product.basic

‚Äč

Retrieve the recommended category for a candidate product based on its title, description, and images.‚Äč

If you are syncing product catalogs from an external system to TikTok Shop, use this API to facilitate product categorization.‚Äč

Note: The language used in text fields such as descriptions and titles must align with the target market's language (e.g. don't use Chinese).‚Äč

‚Äč

Recommendations: ‚Äč

\- If you are syncing a pre-existing description from another platform, include the full HTML source description here.‚Äč

\- Provide a detailed description, ideally over 300 characters.‚Äč

\- Include 3-5 key selling points, each under 250 characters, with supporting images.‚Äč

\- Use 1600x1600 px for the image dimensions.‚Äč
