Create Target Collaboration

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.affiliate\_collaboration.write

вЂ‹

This API is used to create a target collaboration.вЂ‹

A target collaboration is a collaboration between a seller selected set of products (including a commission payout) and a set of creators the seller has added (invited) to the collaboration. Target collaborations are private and not visible in the Creator Marketplace to all creators; they are only visible to those that have been added to the collaboration.вЂ‹
