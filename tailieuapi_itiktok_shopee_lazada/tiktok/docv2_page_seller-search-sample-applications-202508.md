вЂ‹Shop

Required scope:

This API is provided for sellers to query sample applications based on various criteria such as products, creators, or application statuses. By using this API, sellers can retrieve detailed information about sample application records, including the creator's information and the specifics of each application.вЂ‹

statusвЂ‹

 | 

stringвЂ‹

 | 

The status of sample applications.вЂ‹

The possible enumerated values are:вЂ‹

\- PENDING: The sample application is waiting for the seller's review.вЂ‹

\- AWAITING\_SHIPMENT: The application is approved, and the seller needs to ship the sample.вЂ‹

\- SHIPPED: The sample has been shipped by the seller and is waiting for the creator to receive the package.вЂ‹

\- CONTENT\_PENDING: The creator has received the sample package and is expected to create content.вЂ‹

\- REJECT\_CANCELLED: The sample application has been rejected by the seller.вЂ‹

\- OVERDUE\_CANCELLED: The sample application has expired due to being overdue.вЂ‹

\- UNFULFILL\_CANCELLED: The creator did not fulfill the commitment to create content within the agreed timeframe.вЂ‹

\- DEL\_OPEN\_COLLAB: Open collaboration has been deleted.вЂ‹

\- SELLER\_NOT\_SHIP\_CANCELLED: The seller did not ship the sample within the required timeframe.вЂ‹

\- WITHDRAW\_CANCELLED: The creator withdrew the sample application before the seller approved it.вЂ‹

\- UNFULFILLABLE\_CANCELLED: The application was cancelled due to reasons beyond the creator's control, making it impossible to create content.вЂ‹

\- OPS\_CANCELLED: The application was manually cancelled by operations staff.вЂ‹

\- OPS\_FAILED: The application was marked as failed by operations staff.вЂ‹

\- OPS\_COMPLETED: The application was manually marked as completed by operations staff.вЂ‹

\- COMPLETED: The application is complete, and the creator has posted the content.вЂ‹

This field allows for tracking the status of a sample application throughout its lifecycle, providing visibility into each stage of the process for sellers and creators.вЂ‹
