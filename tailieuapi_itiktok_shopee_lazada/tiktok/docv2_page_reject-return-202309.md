Reject Return

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.return\_refund.basic

вЂ‹

Use this API to reject a buyer's return or refund request. The valid decision depends on the current request type and status. Before calling this API, make sure the request is still awaiting the seller action that matches the decision you send.вЂ‹

 | 

stringвЂ‹

 | 

Seller's decision for the request, available values:вЂ‹

\- REJECT\_REFUND: seller reject a refund-only requestвЂ‹

\- REJECT\_RETURN: seller reject the initial return-and-refund requestвЂ‹

\- REJECT\_RECEIVED\_PACKAGE: seller reject return-and-refund request after the buyer has already shipped or handed over the item.вЂ‹

\- REJECT\_REPLACEMENT: seller reject replacement requestвЂ‹
