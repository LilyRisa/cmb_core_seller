Create Return

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.fulfillment.basic

or

вЂ‹seller.return\_refund.basic

вЂ‹

Use this API to create a seller-initiated aftersales request for an order on behalf of the buyer. Supported request types are REFUND and RETURN\_AND\_REFUND. Before submitting the request, confirm that the order is eligible for the selected aftersales type and return method. After the request is created, use the relevant search, status, or review API to continue the aftersales workflow.вЂ‹
