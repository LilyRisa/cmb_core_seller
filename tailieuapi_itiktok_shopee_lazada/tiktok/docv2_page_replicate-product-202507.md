inventory\_modeвЂ‹

 | 

stringвЂ‹

 | 

Defines the inventory mode for the duplicated source product. This parameter supports two modes: SHARED and EXCLUSIVE.вЂ‹

вЂ‹

\- SHARED: In this mode, the inventory of all interconnected (shared) products must be strictly consistent; otherwise, an error will be returned.вЂ‹

\- EXCLUSIVE: In this mode, the inventory for each product can be set independently without affecting others.вЂ‹

вЂ‹

Applicability:вЂ‹

This feature is applicable ONLY for the EU market.вЂ‹

вЂ‹

Default Value:вЂ‹

If left empty, the system defaults to the EXCLUSIVE mode.вЂ‹
