Create Image Translation Tasks

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹serller.product.optimize

вЂ‹

Initiates a translation task for one or more images. This asynchronous API queues the images for processing in the specified target language. вЂ‹

Applicable only for sellers that sell across EU.вЂ‹

Use the [Image Translation Completed webhook](684f8d9bcc4e44049347a12e) to monitor the translation progress, or pass the task ID to the [Get Image Translation Tasks API](684f8b2d535a9d048f234564) to retrieve the status and the translated images.вЂ‹

Note: вЂ‹

\- Please wait 5 minutes after task creation before calling the Get endpoint, and retry at intervals over 10 seconds to avoid rate limiting.вЂ‹

\- Each task corresponds to the translation of 1 image into 1 target language.вЂ‹

\- You can create up to 20 tasks per call and 500 tasks per day. Any tasks beyond the daily limit will be rejected.вЂ‹
