вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.fulfillment.basic

Use this API to ship orders (purchase labels). This API is region specific to the US and JP. The shipping fee and delivery time is an estimate only and is based on the package dimensions and weight you provided. Based on the package attributes, options listed below may differ from your shipping subscriptions.вЂ‹
