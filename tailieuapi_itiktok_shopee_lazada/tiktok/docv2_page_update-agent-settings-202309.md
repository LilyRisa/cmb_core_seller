Update Agent Settings

‚ÄčShop

‚Äč

Required scope:

‚Äčseller.customer\_service

‚Äč

Use this API to update agent status on behalf of the agent. Agents can set whether they can accept auto assigned chats. Before using API, we recommend setting can\_accept\_chat to true, in order to accept chats. Note: 1. The current API version can only update settings on behalf of the shop, or the owner account holder on Seller Center. The owner in this case is acting as the customer service agent. In the future, we plan to make this API available to subaccount holders (who has customer service role) in Seller Center. 2. This API is to allow the agent to update their own setting. See more information in API overview ‚Äč
