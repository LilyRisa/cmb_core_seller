Update Shipping Info

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.logistics

вЂ‹

If the seller entered an incorrect tracking number, this API allows the seller to update the tracking number and shipping provider for an order that has already been shipped.вЂ‹

вЂ‹

\- This API is only applicable to orders (or packages) shipped by the seller. вЂ‹

\- It is only used to update the tracking number and shipping provider for packages that have already been shipped. вЂ‹

\- For orders that have been split for shipping, please use the [Update Package Shipping Info API](https://partner.tiktokshop.com/docv2/page/650aa666c16ffe02b8f1203c?external_id=650aa666c16ffe02b8f1203c). вЂ‹

вЂ‹

Please note that TikTok Shop only allows merchants to update shipping information within 72 hours after shipping.вЂ‹
