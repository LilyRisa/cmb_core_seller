# For TikTok Shop Partners (TSPs)

As a TSP, you may contact support on behalf of a seller. In the **Help Center**, select the option to **Get support for a shop**:  
![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/a12d5b6d2c734a62ae3d9885a5ad0ab7~tplv-k9wyc2ijk0-image.image)  
You will be prompted with the following set of drop-down menus:  
![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/250aa39e3e8847e8b29b1b2ffebda63f~tplv-k9wyc2ijk0-image.image)

1.  For **Target Market**, select the United States.
2.  For **Service**, select your correct TSP service.
3.  For **Shop ID**, select the correct seller from the drop-down menu, or manually input the corresponding Shop ID.

❗ **Important**: If you can't find the correct Shop ID, the shop has most likely not authorized you to contact support on their behalf. Please visit the [FAQs - How can a seller authorize a TSP to contact support](66688ae10664fe02c1da888e#FAQs) at the bottom of this guide.

1.  Select **Start Chat**
2.  A new chat window will open to connect with a seller support agent:

![](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/8eb1f68dc5e84758bed1642507ca4d6c~tplv-k9wyc2ijk0-image.image)

# For Creator Agency Partners (CAPs)/Multi-Channel Networks (MCNs)

In the **Help Center**, select the option to **Get support for a creator**:  
![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/c6a4e38d6da249b9b81d44d68a01665c~tplv-k9wyc2ijk0-image.image)  
You will be prompted with the follow set of drop-down menus:  
![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/97f474ecbe054e85bd589e8cb1914d37~tplv-k9wyc2ijk0-image.image)

1.  For **Target Market**, select the United States.
2.  For **Creator** **Username**, select the correct creator.
3.  Select **Start Chat**.
4.  A new chat window will open to connect with a creator support agent:

![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/2e244a79acdc45fdafdf33673a5bd023~tplv-k9wyc2ijk0-image.image)

# FAQs

## How can a seller authorize a TSP to contact support on their behalf?

1.  The TSP should navigate to **Partner Center** > **Apps and Services** > **Copy authorization link**:

![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/178b708a02d845649777fca017123a28~tplv-k9wyc2ijk0-image.image)

1.  Send the authorization link to the seller. If they are a new seller, ensure they check the box to **Submit a ticket for me** during authorization:

![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/abf49e7a78f749949c8f5f80eaa8f578~tplv-k9wyc2ijk0-image.image)  
Existing sellers will be presented with an authorization renewal prompt, where they will be able to check the same box to **Submit a ticket for me**:  
![Image](https://p16-arcosite-sg.ibyteimg.com/tos-alisg-i-k9wyc2ijk0-sg/2b71bae619134851929f8bc23f3c36f5~tplv-k9wyc2ijk0-image.image)
