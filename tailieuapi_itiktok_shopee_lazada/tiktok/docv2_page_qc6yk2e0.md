# For Vietnam market: Updates to Mandatory attributes on Beauty & Personal Care, Health, Baby & Matern

> Scraped: 2026-05-21T00:46:53.336Z

# What is changing?

Starting from 26 May 2026, partners will be required to provide additional mandatory product attributes when listing or editing products by using API for products under the following categories: Beauty & Personal Care, Health, Baby & Maternity, and Food & Beverages.  
The mandatory product attributes will include :

-   product license details
-   Product License Type
-   Product License Number
-   Date of Issuance
-   Place of Issuance

They will be added into "product\_attributes" parameter.

# Which markets are affected?

The updates of the requirements apply to the local and cross-border sellers in the Vietnam market.

# Who is affected?

Developers with applications that use [Create Product](https://partner.tiktokshop.com/docv2/page/create-product-202309), [Edit Product](https://partner.tiktokshop.com/docv2/page/edit-product-202509), and [Partial Edit Product](<https://partner.tiktokshop.com/docv2/page/partial-edit-product-202509#Back To Top>)

# Which version is applicable?

The updates of the requirements apply to all versions of the related APIs

# What action is required?

Without this mandatory attribute sellers will not be able to create and update products.  
Use the Get Attributes to check what attributes are mandatory and pass attribute values via the Create Product, Edit Product, and Partial Edit Product to help sellers fill in the attribute values.
