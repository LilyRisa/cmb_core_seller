| 

conversation\_status​

 | 

string​

 | 

Filter the list of IM conversations returned based on the status. Possible values: ​

\- ALL: Retrieve all conversations regardless of the status.​

\- UNREAD: Retrieve all conversations with messages that the seller has not read yet. The messages stay unread until explicitly marked as read by calling [Mark Conversation Read](6791db761882810314a991a8). Replying to a message does not change the read / unread status.​

\- UNREPLIED: Retrieve all conversations in which the seller is not the last message sender.​

This filter helps you to efficiently manage and prioritize conversations.​
