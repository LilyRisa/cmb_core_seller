# Overview

> Source: https://partner.tiktokshop.com/docv2/page/tts-api-sdk-overview
> Section: Developer Guide
> Scraped: 2026-05-21T00:25:37.724Z

---

TikTok Shop API SDKs reduces the amount of work required to use TikTok Shop APIs.  
This SDK provides support for the backends of your TikTok Shop apps to access TikTok Shop API, by making it easier to make requests without writing codes for authentication. Currently, you cannot register / process webhooks with this SDK.  
The SDKs are available for Java, Go, and Node.js.  
The SDK is available to part of the TikTok Shop developers currently, and will soon be available for all developers. Check the [console](https://partner.tiktokshop.com/openlearn/api-sdk) to see if you are on the whitelist. If you are, you can access the page like this:
