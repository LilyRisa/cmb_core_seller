Create Responsible Person

вЂ‹Seller

вЂ‹

Required scope:

вЂ‹seller.product.basic

вЂ‹

Add a new EU responsible person who ensures a seller's products comply with EU regulations. The provided information will be automatically translated into all EU languages supported by TikTok Shop.вЂ‹

\- Use the [Search Responsible Persons API](67066a55f17b7d02f95d2fb1) to obtain the translations.вЂ‹

\- Use the [Partial Edit Responsible Person API](67066a5587019802fdce19b3) to edit the translations, if necessary.вЂ‹

\- When creating a product, pass the returned responsible\_person\_id to associate the product with the responsible person.вЂ‹

вЂ‹

Target seller: Local sellers operating in EU countriesвЂ‹
