Get Payments

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.finance.info

вЂ‹

This API is currently unavailable to SEA markets.вЂ‹

Retrieves records of automated payments for a shop based on a specified date range.вЂ‹

Use the returned list to verify and reconcile payments with the transactions in the seller's bank account.вЂ‹
