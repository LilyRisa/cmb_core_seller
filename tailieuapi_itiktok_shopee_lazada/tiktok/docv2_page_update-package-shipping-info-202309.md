Update Package Shipping Info

‚ÄčShop

‚Äč

Required scope:

‚Äčseller.fulfillment.basic

‚Äč

If the seller entered an incorrect tracking number, this API allows the seller to update the tracking number and shipping provider for a package that has already been shipped. Attention: This API is only applicable to orders (or packages) shipped by the seller. It is only used to update the tracking number and shipping provider for packages that have already been shipped.‚Äč
