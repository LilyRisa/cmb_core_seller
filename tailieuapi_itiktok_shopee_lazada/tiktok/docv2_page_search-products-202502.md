вЂ‹Shop

listing\_platformsвЂ‹

 | 

\[\]stringвЂ‹

 | 

Filter products by the listing platforms.вЂ‹

Possible values:вЂ‹

\- TOKOPEDIAвЂ‹

\- TIKTOK\_SHOPвЂ‹

Default: Return all products regardless of their listing platform.вЂ‹

вЂ‹

Applicable only for sellers that migrated from Tokopedia.вЂ‹

Note:вЂ‹

\- You must also specify a status value other than ALL when filtering by listing platforms. Returning all statuses is not supported.вЂ‹

\- If you pass in one platform, the search will return products that are listed on that platform, including those that are listed on both platforms.вЂ‹

\- If you pass in \["TIKTOK\_SHOP", "TOKOPEDIA"\], only products listed on both platforms will be returned, not those listed on just one.вЂ‹

 |

 | 

\[\]stringвЂ‹

 | 

Filter products by their audit status for TikTok Shop.вЂ‹

Possible values: вЂ‹

\- FAILED: Returns products where the base version or a post-live edit has failed audit, or had the audit cancelled.вЂ‹

\- APPROVED: Returns products that passed the audit and has been listed on the platform.вЂ‹
