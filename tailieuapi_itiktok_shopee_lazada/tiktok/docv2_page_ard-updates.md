# Summary

The [Connector ARD](connector-multi-channel-dropshipping-and-print-on-demand) has been updated to include a few new use cases, like:

-   On Hold Order Status
-   Get Warehouse API Calls
-   Toggle Options for Product fields
-   Listing Schemas for Categories
-   Check Prerequisites before product listing

Additionally, a new [Customer Service ARD](app-features) has also been published.
