‚ÄčShop

‚Äč

Required scope:

Create an engagement task that acts as a container for grouping messages with similar content and rules, allowing sellers to track and compare task performance across different types of content.‚Äč

Note that each task has a mandatory end time, and once expired, it cannot be used to send additional messages.‚Äč
