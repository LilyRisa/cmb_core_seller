Get Global Categories

вЂ‹Seller

вЂ‹

Required scope:

вЂ‹seller.global\_product.category.info

вЂ‹

Retrieve all available product categories, regardless of market variations.вЂ‹

Product categories are updated frequently, so it's recommended to call the API in real time to ensure you are using the latest category data. Caching category data locally may result in using outdated information, leading to errors when creating global products. вЂ‹
