# 1\. Trigger scenario

This webhook is triggered when a product package recommendation is generated.  
**Note:** Applicable only for Brazil (**BR**) and Mexico (**MX**) markets.  
**Prerequisite**: The "Product Basic" API scope is enabled in Partner Center. For more information, please refer to [Access Scope](https://partner.tiktokshop.com/docv2/page/access-scope).
