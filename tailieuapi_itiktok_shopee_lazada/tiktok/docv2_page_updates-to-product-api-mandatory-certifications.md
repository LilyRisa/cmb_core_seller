# Updates to Product API Mandatory Certifications

> Source: https://partner.tiktokshop.com/docv2/page/updates-to-product-api-mandatory-certifications
> Section: Changelog
> Scraped: 2026-05-21T00:39:35.335Z

---

# What is changing?

TikTok Shop is implementing new Mandatory Certifications for certain categories.  
Starting from **2025-02-21**, Product APIs will be updated to require Mandatory Certifications for the following categories:

-   **COA for Collectibles** and **Certificate for Autograph Collectibles** will be required for the Collectibles category
    
-   **Test reports** and **Tracking info labelling** will be required for the Toys & Hobbies category
    

# Which markets are affected?

The updates of the requirements apply to local sellers in the US market

# Who is affected?

Developers with applications that use Create Product, Edit Product, and Partial Edit Product

# Which version is applicable?

The updates of the requirements apply to all versions of the related APIs

# What action is required?

Without mandatory product certifications, sellers will not be able to create and update products within specific categories  
Use the [Get Category Rule](get-category-rules) to check what product certifications are required for which categories, and pass the product certification files via the Create Product, Edit Product, and Partial Edit Product to help sellers submit product certification.
