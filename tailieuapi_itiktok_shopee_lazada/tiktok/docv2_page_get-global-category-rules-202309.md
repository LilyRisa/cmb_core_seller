Get Global Category Rules

‚ÄčSeller

‚Äč

Required scope:

‚Äčseller.global\_product.category.info

‚Äč

Retrieve the additional requirements (beyond mandatory product attributes) for listing a global product in a particular category, regardless of market variations. Requirements may include product certifications, size charts, dimensions and more.‚Äč

Use this API to determine the supporting information that you must prepare before listing a global product.‚Äč

Note: It must be a [leaf category](650a03f8f1fd3102b91b338a) that corresponds to the category tree type specified in the category\_version property.‚Äč
