# ID market: Tokopedia Open API Termination - Migrate to TikTok Shop Partner Center by September 30, 2

> Source: https://partner.tiktokshop.com/docv2/page/k1156vkq
> Section: Changelog
> Scraped: 2026-05-21T00:51:20.242Z

---

The **complete termination of** [Tokopedia Open API](https://developer.tokopedia.com/) **services** will be effective on **September 30, 2025**. As part of our strategic integration with the broader TikTok Shop ecosystem, we are transitioning all partner integrations to  
[TikTok Shop Partner Center by Tokopedia](https://partner.tiktokshop.com/)  
We strongly encourage all partners to immediately migrate to TTS Partner Center, where you can manage both Tokopedia and TikTok Shop operations seamlessly using a single, unified API platform. This migration will provide you with:

-   **Unified Management:** Control both your Tokopedia and TikTok Shop stores from one API console
    
-   **Enhanced Features:** Access to advanced developer tools and analytics (e.g. SDKs, API logs, Load Test Tools)
    
-   **Streamlined Operations:** Reduce complexity by managing multiple storefronts through one API
    

**TERMINATION TIMELINE (3 Phases)**  
  
**PHASE 1 - Tokopedia Developer Portal & Documentation Closure**  
This phase focuses on closing all developer resources and documentation portals. We are providing alternative access points to ensure continuity for partners who migrate to TikTok Shop Partner Center.

-   Deadline: **September 10, 2025**
-   Services Terminated:
    -   Tokopedia Developer Portal
    -   API documentation and guideline
    -   Developer registration page
    -   News & update page
    -   FAQ page
-   Immediate action required:
    -   Download and save all necessary API documentation locally
        
    -   Export your app credentials and configuration settings
        
    -   Register immediately at TikTok Shop Partner Center
        
    -   Update bookmarks to new redirect URLs (see table below)

**PHASE 2 - Tokopedia Developer Console & API Modification Termination**  
This phase eliminates all data modification capabilities and developer management tools. Only read-only operations will remain functional until Phase 3.

-   Deadline: **September 20, 2025**
-   Services Terminated:
    -   Developer Console access
    -   All POST & PATCH API endpoints
    -   Webhook registration and management
    -   App credential management system
    -   API testing tools and sandbox environment
-   Action required:
    -   Complete migration to TikTok Shop Partner Center before this date
        
    -   Cease all data modification operations (POST/PATCH requests)
        
    -   Complete final webhook configurations through TTS console before deadline
        

**PHASE 3 - Complete API & Webhook Termination**  
This final phase marks the complete shutdown of all Tokopedia API services. After this date, no Tokopedia API functionality will be available.

-   Deadline: **September 30, 2025**
-   Services Terminated:
    -   All remaining API endpoints (GET methods)
    -   All webhook services
    -   Authentication and authorization functions
    -   Legacy support channels
-   Final action required:
    -   Completely discontinue all Tokopedia API integrations
        
    -   Ensure full migration to TikTok Shop Partner Center is completed
        
    -   Remove Tokopedia API dependencies from your applications
        
    -   Update your applications to prevent API call failures
