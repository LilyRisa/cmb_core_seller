Recommend Global Categories

вЂ‹Seller

вЂ‹

Required scope:

вЂ‹seller.global\_product.category.info

вЂ‹

Retrieve the recommended categories for a candidate global product based on its title, description, and images.вЂ‹

If you are syncing product catalogs from an external system to TikTok Shop, use this API to facilitate product classification.вЂ‹

Note: Double-byte characters (e.g. Chinese characters) are not supported in text fields such as descriptions and titles. If you include them, the API request will fail.вЂ‹

\- If you are syncing a pre-existing description from another platform, include the full HTML source description here.вЂ‹

\- Provide a detailed description, ideally over 300 characters.вЂ‹

\- Include 3-5 key selling points, each under 250 characters, with supporting images.вЂ‹

\- Use 1600x1600 px for the image dimensions.вЂ‹
