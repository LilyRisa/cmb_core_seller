Create Activity

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.promotion.write

вЂ‹

Use this API to create a product discount activity or a flash deal activity. You cannot create a coupon activity with this API. To do that, you'll need to go to the seller center or the seller app. To update the products or SKUs included in the activity, use [Update Activity Product](https://partner.tiktokshop.com/docv2/page/650d32c42aaa3602b86ccb5c).вЂ‹

activity\_typeвЂ‹

RequiredвЂ‹

 | 

stringвЂ‹

 | 

The type of activities.вЂ‹

Possible enumerations:вЂ‹

\- FIXED\_PRICE: product discount activity with fixed price.вЂ‹

\- DIRECT\_DISCOUNT: product discount activities with percentage off.вЂ‹

\- FLASHSALE: flash sale activity.вЂ‹

\- SHIPPING\_DISCOUNT: shipping fee discount.вЂ‹

\- BUY\_MORE\_SAVE\_MORE: Buy more save more discount.вЂ‹

\- BUY\_X\_GET\_Y: Buy X get Y.вЂ‹

 |
| 

product\_levelвЂ‹

RequiredвЂ‹

 | 

stringвЂ‹

 | 

Activity product dimension, values are:вЂ‹

вЂ‹- PRODUCT: Promotion activity applies at the product (SPU) level. Any and all SKUs that are part of this product will be subject to the promotion.вЂ‹

вЂ‹- VARIATION: Promotion activity only applies to specified SKUs.вЂ‹

вЂ‹- SHOP: Promotion activity applies to all products in the shop.вЂ‹

When activity\_type is SHIPPING\_DISCOUNT, the product\_level can not be VARIATION.вЂ‹

To set the products or SKUs covered by the activity, use [Update Activity Product](https://partner.tiktokshop.com/docv2/page/650d32c42aaa3602b86ccb5c).вЂ‹

 |
| 

duration\_typeвЂ‹

 | 

stringвЂ‹

 | 

The effective time type of the current activityвЂ‹

\- NORMAL: begin\_time and end\_time must be filledвЂ‹

\- INDEFINITE: begin\_time and end\_time aren't required, and the promotion will be a long-term activity which runs indefinitely. вЂ‹

1\. INDEFINITE is only valid when activity\_type is SHIPPING\_DISCOUNT,вЂ‹

2\. If duration\_type is INDEFINITE, start\_time and end\_time should be 0вЂ‹

Default: NORMALвЂ‹
