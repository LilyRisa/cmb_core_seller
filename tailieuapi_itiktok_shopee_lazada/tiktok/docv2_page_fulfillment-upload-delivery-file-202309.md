Fulfillment Upload Delivery File

вЂ‹Shop

вЂ‹

Required scope:

вЂ‹seller.fulfillment.basic

вЂ‹

This API is used for the seller to upload the proof of delivery file for a package, and to generate the URL of the corresponding file. The generated URL is used for the [Update Package Delivery Status API](https://partner.tiktokshop.com/docv2/page/650aa332c16ffe02b8f0ba82?external_id=650aa332c16ffe02b8f0ba82). вЂ‹

вЂ‹

This API only supports uploading qualification files in PDF format. The file size can not exceed 10MB.вЂ‹

вЂ‹

Note: Only sellers utilizing the SOF (Seller Own Fleet) capability can use this API. вЂ‹
