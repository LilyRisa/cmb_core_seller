# Ad Comments - Blocked Words

Blocked words are used to automatically hide comments. If a comment contains a blocked word, it will be hidden to everyone except the user who made the comment. 

This group consists of the following endpoints:
* [Create blocked words](https://ads.tiktok.com/marketing_api/docs?id=1739029218470913)
* [Update a blocked word](https://ads.tiktok.com/marketing_api/docs?id=1739029234526209)
* [Check the statuses of words](https://ads.tiktok.com/marketing_api/docs?id=1739029247961089)
* [Get blocked words](https://ads.tiktok.com/marketing_api/docs?id=1739029260837889)
* [Delete blocked words](https://ads.tiktok.com/marketing_api/docs?id=1739029277346817)
* [Create an export task (blocked words)](https://ads.tiktok.com/marketing_api/docs?id=1739029296119809)
* [Check the status of an export task (blocked words)](https://ads.tiktok.com/marketing_api/docs?id=1739029311687681)
* [Download exported data](https://ads.tiktok.com/marketing_api/docs?id=1739029325203457)
