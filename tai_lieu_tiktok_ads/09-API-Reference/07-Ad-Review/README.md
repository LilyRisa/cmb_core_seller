# Ad Review

Ad Review API enables you to get the review information for ad groups and ads, including the review status, rejection reasons, and suggestions, or to request for a re-evaluation after rejection.

This group consists of the following endpoints:
* [Get review info of ad groups](https://ads.tiktok.com/marketing_api/docs?id=1740554277061634)
* [Get review info of ads](https://ads.tiktok.com/marketing_api/docs?id=1740559669226498)
* [Appeal a rejection](https://ads.tiktok.com/marketing_api/docs?id=1740571821728770)
