# Download the invoices of a BC

Use these endpoints to download invoices, via synchronous downloads or asynchronous downloads. 

Both Agency and Direct Business Centers are able to perform invoice related operations.   

> <span style="color: dodgerblue">**Note**</span><br>Only users with Finance Manager or Finance Analyst permission can manage payments and invoices in Business Center. You can use [/bc/member/invite/](https://ads.tiktok.com/marketing_api/docs?id=1739939455765505) and [/bc/member/update/](https://ads.tiktok.com/marketing_api/docs?id=1739696704424961) to configure or update access settings for members.

This group consists of the following endpoints: 
- [Download synchronously](https://ads.tiktok.com/marketing_api/docs?id=1754000198430722)
- [Download asynchronously](https://ads.tiktok.com/marketing_api/docs?id=1754000224141313)
  - [Create an asynchronous download task](https://ads.tiktok.com/marketing_api/docs?id=1754000448122882)
  - [Get asynchronous download tasks](https://ads.tiktok.com/marketing_api/docs?id=1754000480307265)
  - [Get asynchronous download task list](https://ads.tiktok.com/marketing_api/docs?id=1754000504437762)
