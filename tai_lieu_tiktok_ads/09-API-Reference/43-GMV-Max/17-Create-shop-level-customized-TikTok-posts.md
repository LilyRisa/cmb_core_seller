# Create shop-level customized TikTok posts

Use this endpoint to create shop-level customized TikTok posts.

Shop-level customized TikTok posts are generated by directly linking your TikTok videos to a product at the TikTok Shop level. If the linked product is part of an active campaign, the video linked to that product will be automatically added to that campaign.


## Request
**Endpoint** https://business-api.tiktok.com/open_api/v1.3/gmv_max/creation/custom_anchor_video_list/create/


**Method** POST

**Header**

```xtable
|Field{35%}|Type{15%}|Description{50%}|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://business-api.tiktok.com/portal/docs/marketing-api-authentication/v1.3).|
|Content-Type {Required}|string|Request message type.<br>Allowed value: `application/json`.|
```

**Parameters**

```xtable
|Field{35%}|Data Type{15%}|Description{50%}|
|---|---|---|
|advertiser_id {Required}|string|Advertiser ID.|
|store_id {Required}|string|The ID of a TikTok Shop.<br><br>To obtain a TikTok Shop that is available for GMV Max Campaigns, use [/gmv_max/store/list/](https://business-api.tiktok.com/portal/docs?id=1822001044479041) and confirm that the returned `is_gmv_max_available` is `true`.|
|store_authorized_bc_id {Required}|string|The ID of the Business Center that is authorized to access the TikTok Shop (`store_id`).<br><br>To obtain the ID, check the `store_authorized_bc_id` for the `store_id` returned from [/gmv_max/store/list/](https://business-api.tiktok.com/portal/docs?id=1822001044479041).|
|custom_anchor_video_list {Required}|object[]|The details of your videos and the products you want to link them to.<br><br>By linking videos to a product at the TikTok Shop level, you can generate shop-level customized TikTok posts. The system will automatically add these shop-level customized TikTok posts to your Product GMV Max Campaign (with `shopping_ads_type` set to `PRODUCT`) if you use the Autoselect creative mode (with `product_video_specific_type` as `AUTO_SELECTION`) and your campaign includes the linked product.<br><br>Max size: 200.|
#|item_id {Required}|string|The ID of the TikTok video you intend to use in the customized post.<br><br>To retrieve eligible videos, use [/gmv_max/video/get/](https://business-api.tiktok.com/portal/docs/get-posts-for-a-product-gmv-max-campaign/v1.3) with `custom_posts_eligible` set to `true`.|
#|identity_info {Required}|object|Information about the identity (TikTok account) associated with the TikTok video.<br><br>To find the identity details associated with the provided TikTok video (`item_id`), review the `identity_info` corresponding to the `item_id` returned by [/gmv_max/video/get/](https://business-api.tiktok.com/portal/docs/get-posts-for-a-product-gmv-max-campaign/v1.3).|
##|identity_id {Required}|string|Identity ID.|
##|identity_type {Required}|string|Identity type.<br><br>Enum values:<ul><li>`AUTH_CODE`: Authorized User. This type of identity is created when you use the authorization code to access a TikTok account or a TikTok post.</li><li>`TT_USER`: TikTok User. This type of identity is created when you bind your TikTok For Business account with your TikTok Business Account, or when you bind your TikTok For Business account with your regular TikTok account and then upgrade the account to Business Account.</li><li>`BC_AUTH_TT`: TikTok Account User in Business Center. This type of identity is created when you add a TikTok account to your Business Center, and the TikTok account owner approves your request.</li><li>`TTS_TT`: TikTok Account User for TikTok Shop. This type of identity is created when you set an official TikTok account for the TikTok Shop.</li></ul>|
##|identity_authorized_bc_id {+Conditional}|string|Required when `identity_type` is `BC_AUTH_TT`.<br><br>The ID of the Business Center that a TikTok Account User in Business Center identity is associated with.|
##|identity_authorized_shop_id {+Conditional}|string|Required only when `identity_type` is `BC_AUTH_TT` and `identity_authorized_shop_id` is returned for the identity from [/gmv_max/video/get/](https://business-api.tiktok.com/portal/docs/get-posts-for-a-product-gmv-max-campaign/v1.3).<br><br>The ID of the TikTok Shop that a TikTok Account User in Business Center identity is associated with.|
##|store_id {+Conditional}|string|Required when `identity_type` is `TTS_TT`.<br><br>The ID of the TikTok Shop that a TikTok Account User for TikTok Shop identity is associated with.|
#|spu_id_list {Required}|string[]|The SPU IDs of the products to link to the video.<br><br>Max size: 1.<br><br>To obtain the list of SPU IDs for products within a TikTok Shop, use [/store/product/get/](https://business-api.tiktok.com/portal/docs?id=1793482248880130). Set `ad_creation_eligible` to `GMV_MAX` and select `item_group_id` values where `status` is `AVAILABLE` and `gmv_max_ads_status` is `UNOCCUPIED`.|
```

### Example
Request
```xcodeblock
(code curl http)
curl --location --request POST 'https://business-api.tiktok.com/open_api/v1.3/gmv_max/creation/custom_anchor_video_list/create/' \
--header 'Access-Token: {{Access-Token}}' \
--header 'Content-Type: application/json' \
--data '{
    "advertiser_id": "{{advertiser_id}}",
    "store_id": "{{store_id}}",
    "store_authorized_bc_id": "{{store_authorized_bc_id}}",
    "custom_anchor_video_list": [
        {
            "identity_info": {
                "identity_authorized_bc_id": "{{identity_authorized_bc_id}}",
                "identity_id": "{{identity_id}}",
                "identity_type": "BC_AUTH_TT"
            },
            "item_id": "{{item_id}}",
            "spu_id_list": ["{{spu_id}}"]
        },
        {
            "identity_info": {
                "identity_id": "{{identity_id}}",
                "identity_type": "TT_USER"
            },
            "item_id": "{{item_id}}",
            "spu_id_list": ["{{spu_id}}"]
        }
    ]
}'
(/code)
```
