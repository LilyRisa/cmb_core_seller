# Delete customized TikTok posts

Use this endpoint to delete shop-level or campaign-level customized TikTok posts.

Shop-level customized TikTok posts are generated by directly linking your TikTok videos to a product at the TikTok Shop level. If the linked product is part of an active campaign, the video linked to that product will be automatically added to that campaign. In contrast, campaign-level customized TikTok posts are generated by linking your TikTok videos to a product during the Product GMV Max Campaign creation process and are exclusive to the campaign.


## Request
**Endpoint** https://business-api.tiktok.com/open_api/v1.3/gmv_max/creation/custom_anchor_video_list/delete/


**Method** POST

**Header**

```xtable
|Field{35%}|Type{15%}|Description{50%}|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://business-api.tiktok.com/portal/docs/marketing-api-authentication/v1.3).|
|Content-Type {Required}|string|Request message type.<br>Allowed value: `application/json`.|
```

**Parameters**

```xtable
|Field{35%}|Data Type{15%}|Description{50%}|
|---|---|---|
|advertiser_id {Required}|string|Advertiser ID.|
|store_id {Required}|string|The ID of a TikTok Shop.<br><br>To obtain a TikTok Shop that is available for GMV Max Campaigns, use [/gmv_max/store/list/](https://business-api.tiktok.com/portal/docs?id=1822001044479041) and confirm that the returned `is_gmv_max_available` is `true`.|
|store_authorized_bc_id {Required}|string|The ID of the Business Center that is authorized to access the TikTok Shop (`store_id`).<br><br>To obtain the ID, check the `store_authorized_bc_id` for the `store_id` returned from [/gmv_max/store/list/](https://business-api.tiktok.com/portal/docs?id=1822001044479041).|
|custom_anchor_video_list {Required}|object[]|The details of the videos and the products you want to unlink from them.<br><br>Max size: 200.|
#|item_id {Required}|string|The ID of the TikTok video you no longer want to use in a customized post.<br><br>To retrieve TikTok videos currently used in customized posts, use [/gmv_max/creation/custom_anchor_video_list/get/](https://business-api.tiktok.com/portal/docs/get-customized-tiktok-posts/v1.3).|
#|spu_id_list {Required}|string[]|The Product SPU (standard product unit) ID of the product you want to unlink from the video.<br><br>To retrieve the products linked to a TikTok video used in customized posts, use [/gmv_max/creation/custom_anchor_video_list/get/](https://business-api.tiktok.com/portal/docs/get-customized-tiktok-posts/v1.3).|
|campaign_id {+Conditional}|string|Required when the specified TikTok videos (`item_id`) have been used in a campaign to create campaign-level customized posts.<br><br>The ID of a Product GMV Max Campaign.<br><br>To retrieve existing Product GMV Max Campaigns within your ad account, use [/gmv_max/campaign/get/](https://business-api.tiktok.com/portal/docs?id=1826463372290177) and set `filtering` to `&#123;"gmv_max_promotion_types":["PRODUCT_GMV_MAX"]&#125;`.|
```

### Example
```xcodeblock
(code curl http)
curl --location --request POST 'https://business-api.tiktok.com/open_api/v1.3/gmv_max/creation/custom_anchor_video_list/delete/' \
--header 'Access-Token: {{Access-Token}}' \
--header 'Content-Type: application/json' \
--data '{
    "advertiser_id": "{{advertiser_id}}",
    "store_id": "{{store_id}}",
    "store_authorized_bc_id": "{{store_authorized_bc_id}}",
    "custom_anchor_video_list": [
        {
            "item_id": "{{item_id}}",
            "spu_id_list": ["{{spu_id}}"]
        },
        {
            "item_id": "{{item_id}}",
            "spu_id_list": ["{{spu_id}}"]
        }
    ]
}'
(/code)
```
