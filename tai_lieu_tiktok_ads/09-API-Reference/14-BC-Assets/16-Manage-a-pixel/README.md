# Manage a pixel

Use these endpoints to transfer a pixel to a Business Center, link a pixel to ad accounts or unlink a pixel from ad accounts, and get a list of ad accounts that are linked to a pixel. 

This group consists of the following endpoints: 
- [Transfer a pixel from advertiser to BC](https://ads.tiktok.com/marketing_api/docs?id=1753994928261122)
- [Link a pixel to ad accounts / Unlink](https://ads.tiktok.com/marketing_api/docs?id=1753994950327297)
- [Get ad accounts linked to a pixel](https://ads.tiktok.com/marketing_api/docs?id=1753994966965249)
