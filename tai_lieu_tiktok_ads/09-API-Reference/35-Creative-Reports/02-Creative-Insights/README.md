# Creative Insights

Use this group of endpoints to get the performance data of ads against benchmarks and in-second performance data.

This group consists of the following endpoints:
* [Get ad benchmarks](https://ads.tiktok.com/marketing_api/docs?id=1738824501176321)
* [Get in-second performance](https://ads.tiktok.com/marketing_api/docs?id=1738825259075586)
