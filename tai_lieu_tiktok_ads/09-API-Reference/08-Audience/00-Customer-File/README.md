# Customer File

Customer File API is a file-based API that can be used to build your target custom audience from uploaded data. To learn more about Customer File API and how it differs from Streaming API, see [Customer File API vs Streaming API](https://ads.tiktok.com/marketing_api/docs?id=1739566528222210). 

This group consists of the following endpoints.
* [Upload an audience file](https://ads.tiktok.com/marketing_api/docs?id=1739940567842818) 
* [Create an audience by file](https://ads.tiktok.com/marketing_api/docs?id=1739940570793985)
