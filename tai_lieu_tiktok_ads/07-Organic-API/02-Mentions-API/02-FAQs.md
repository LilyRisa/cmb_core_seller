# FAQs

This article summarizes FAQs about the Mentions API.

# Data availability
## Is there any delay between when a mention is made and when it will be returned via API?
Yes. There is an approximate delay of 24-48 hours between when a mention is made and when it becomes available via the API.

## Is there a minimum number of post mentions required for results to be returned in [/business/mention/video/list/](https://business-api.tiktok.com/portal/docs?id=1825107955294402)?
Yes. For results to be returned in `/business/mention/video/list/`, at least one post must mention the business.
