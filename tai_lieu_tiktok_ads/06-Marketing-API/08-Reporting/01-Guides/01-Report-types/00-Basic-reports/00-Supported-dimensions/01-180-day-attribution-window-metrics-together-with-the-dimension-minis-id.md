# 180-day attribution window metrics together with the dimension minis_id

The following tables list the specific metrics that are calculated using a 180-day attribution window when the dimension `minis_id` is used.

# Core metrics

```xtable
|Field{45%}|Description{55%}|
|---|---|
| `conversion` | Conversions |
| `cost_per_conversion` | Cost per conversion |
| `conversion_rate` | Conversion rate (CVR, clicks) |
```
