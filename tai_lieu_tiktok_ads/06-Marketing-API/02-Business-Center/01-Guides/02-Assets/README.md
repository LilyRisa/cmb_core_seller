# Assets

You can use Business Center API to manage assets or Asset Groups in a Business Center. 
- [Assets](https://ads.tiktok.com/marketing_api/docs?id=1754705407307777): Assets include ad accounts, catalogs, pixels, leads, TikTok accounts and stores. 
- [Asset Groups](https://ads.tiktok.com/marketing_api/docs?id=1754705410074626): An Asset Group is a collection of assets in the Business Center.
