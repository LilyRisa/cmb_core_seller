# Create a Lookalike Audience

This article introduces how to create Lookalike Audiences.

To create a lookalike audience, call [/dmp/custom_audience/lookalike/create/](https://ads.tiktok.com/marketing_api/docs?id=1739940511844353) and provide an existing audience under your advertiser account. 

If your call is successful, the response will return a `custom_audience_id`, which can be used to update, get or delete the audience.
