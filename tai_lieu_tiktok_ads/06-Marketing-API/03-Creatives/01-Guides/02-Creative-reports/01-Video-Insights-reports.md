# Video Insights reports

Use [/creative/report/get/](https://business-api.tiktok.com/portal/docs?id=1789700050715650) to run a Video Insights report for multiple videos across one or more ad accounts, encompassing both Spark Ads and non-Spark Ads.

To find out the Video Insights metrics supported in the API, see [Supported metrics in Video Insights reports](https://business-api.tiktok.com/portal/docs?id=1789703614085122).
