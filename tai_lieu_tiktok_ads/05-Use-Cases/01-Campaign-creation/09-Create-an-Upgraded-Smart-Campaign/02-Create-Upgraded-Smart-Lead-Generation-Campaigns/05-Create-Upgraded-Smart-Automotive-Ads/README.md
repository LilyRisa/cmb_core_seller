# Create Upgraded Smart+ Automotive Ads

This section walks you through the steps to create Upgraded Smart+ Automotive Ads.

Based on your preference for the catalog type, you can create Upgraded Smart+ Automotive Ads for Inventory using an Auto-Inventory catalog or Upgraded Smart+ Automotive Ads for Models using an Auto-Model catalog. For detailed instructions on creating each type, refer to their respective articles: 
- [Create Upgraded Smart+ Automotive Ads for Inventory](https://business-api.tiktok.com/portal/docs?id=1843325008693250)
- [Create Upgraded Smart+ Automotive Ads for Models](https://business-api.tiktok.com/portal/docs?id=1843325022799873)
